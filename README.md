# chris tech hrms

Multi-tenant HR Management SaaS platform built by Chris Tech (Zetu Business Solutions).

## Overview

This repository contains the source code for **chris tech hrms**, a project developed and maintained by Chris Odhiambo (learninghub44). It is part of a portfolio of production and in-progress software products, several of which target the Kenyan and East African market.

## Tech Stack

- Primary language: TypeScript
- Topics: chris-tech, express, groq-ai, hrms, multi-tenant, nextjs, postgresql, prisma, saas, typescript

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/learninghub44/chris-tech-hrms.git
   cd chris-tech-hrms
   ```
2. Install dependencies (see `package.json`, `requirements.txt`, or equivalent manifest in this repo for the exact commands).
3. Configure environment variables as required by the project (see `.env.example` if present).
4. Run the project locally using the appropriate start/dev script for this stack.
